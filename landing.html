// what's inside

Everything you need
to study computation.

01 📼
Infinite Tape Visualizer

Watch the read/write head traverse the tape in real time. Active cells are highlighted with smooth animations and the head pointer stays pixel-perfect.

02 🔵
Live State-Space Graph

The transition diagram updates as you edit rules. Active states and edges glow as the machine runs, giving you spatial intuition for the computation.

03 📋
Full Execution Trace

Every transition is logged: current state, symbol read, symbol written, direction, and next state — so you can audit every decision the machine makes.

04 ⚙️
Editable Rule Table

Write rules in plain CSV syntax: state, read, write, dir, next. Reload instantly and the graph re-renders to reflect your changes.

05 ▶️
Run / Step / Pause

Run continuously at adjustable speed, step through one transition at a time, or pause mid-computation to inspect state. You control the clock.

06 🎓
Zero Setup

Pure HTML/CSS/JS — no build tools, no dependencies. Open in a browser and start experimenting. Ideal for classrooms and self-study.

// workflow

From rules to
running machine.

# state, read, write, dir, next
q0, 1, 0, R, q0
q0, 0, 1, R, q0
q0, B, B, L, halt

# multi-state example
q0, 0, 0, R, q1
q1, 1, 1, L, q0
q1, B, B, R, halt
01
Write your transition rules
Define each rule as a CSV row: current state, symbol to read, symbol to write, direction (L/R), and next state.
02
Load your input string
Type a binary string (or any symbols) into the input field. The tape initializes with blank padding on each side.
03
Run, step, or pause
Watch the tape and graph animate in sync. Read the execution trace to understand every transition taken.
