02 — Features
Everything you need
to study computation.
F — 01
Infinite Tape Visualizer

Watch the read/write head traverse in real time. The active cell glows with smooth, weighted transitions and the tape auto-extends at boundary.

F — 02
Live State Diagram

Powered by vis-network. The transition graph is interactive — drag nodes, zoom in, watch active states glow as the machine runs.

F — 03
Execution Trace

Every transition is logged — state, symbol read, symbol written, direction, and next state — so you can audit every decision taken.

F — 04
Glassmorphic Editor

Write rules in plain CSV syntax: state, read, write, dir, next. Reload instantly; graph re-renders.

F — 05
Run · Step · Pause

Run continuously, step one transition at a time, or pause mid-computation to inspect state. Adjustable speed slider included.

F — 06
Zero Dependencies

Pure HTML, CSS, JS — no build tools, no package managers. Open in a browser and start experimenting. Ideal for classrooms.

01
Fill out the Transition Table

Enter each rule sequentially into the Transition Table grid, formatting variables as lowercase (q0, q1, q_accept).

02
Load your input string

Type any binary string into the input field. The tape initializes with blank padding on each side.

03
Run, step, or pause

Watch the tape and state graph animate in sync. Read the execution trace to understand every transition.

04
Explore and iterate

Edit rules live, re-load the tape, and observe how the machine's behavior changes. Perfect for building intuition.
